**New tab**
<br />Control + Shift + N

**Save file**
<br />Command + S

**Refresh directory tree**
<br />Control + Shift + R

**Close tab**
<br />Control + Shift + W

**Select next tab**
<br />Control + Shift + ]

**Select previous tab**
<br />Control + Shift + [

**Focus console**
<br />Control + Shift + C

**Toggle full-screen console**
<br />Control + Shift + F
